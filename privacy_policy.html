Information Collection and Use

The Application uses the camera and gallery solely for the purpose of taking and selecting photos. The images captured or selected from the gallery are used within the application and may be stored on our backend servers.

Camera Permission

The Application requests permission to use the camera to allow users to take photos directly from the app which will be used to automatically create item suggestions from a given bill. This permission is used exclusively for the purpose of providing this functionality within the Application.

Gallery Permission

The Application requests permission to access the gallery to allow users to select and upload images from their device. This permission is used exclusively for the purpose of providing this functionality within the Application.
